\newacronym[description={\sloppy%
A \sql\ injection attack is a web attack that is used to target data stored in \dbms\ by injecting malicious input into code that constructs \sql\ queries by string concatenation in order to subvert application functionality and perform unauthorized operations~\cite{ASO2024SIADPP,KDP2024AOSIAITCAIWA,CMCVGHRDCAAFL2023SIADINFD,databases}. %
In order to prevent such attacks, queries to \pglspl{db} should \emph{never} be constructed via string concatenation or the likes of \python\ \glspl{fstring}. %
Assume that \pythonil{user_id} was a string variable in a \python\ program and we construct the query \pythonil{f"SELECT * FROM data WHERE user_id = \{user_id\}"}. %
Notice that the \textil{\{user_in\}} will be replaced with the value of variable~\pythonil{user_id} during \gls{strinterpolation}. %
If \pythonil{user_id == "user123; DROP TABLE data;"}, mayhem would ensue when we execute the query~\cite{PEP675}. %
Some programming languages, like \python, offer built-in datatypes (such as \pythonilIdx{LiteralString}~\cite{PEP675}) to annotate string constants that can be used by static type-checkers. %
At the time of this writing, \mypy\ does not support this yet~\cite{ZDWVSLS2022I1SP6L,VDGE2022PPDAFP:ST}.%
}]{SQLi}{SQLi attack}{\glslink{SQL}{SQL} injection attack}%

\newacronym[description={%

It is the text stream to which the process writes information about errors and exceptions. %
If an uncaught \pythonilIdx{Exception} is raised in \python\ and the program terminates, then this information is written to \gls{stderr}. %
If you run a program in a \gls{terminal}, then the text that a process writes to its \gls{stderr} appears in the console.%
},sort={stderr}]{stderr}{\texttt{stderr}}{standard error stream}%

It is the text stream from which the process reads its input text, if any. %
The \python\ instruction \pythonilIdx{input} reads from this stream. %
If you run a program in a \gls{terminal}, then the text that you type into the terminal while the process is running appears in this stream.%
},sort={stdin}]{stdin}{\texttt{stdin}}{standard input stream}%

The \pythonilIdx{print} instruction of \python\ writes text to this stream. %
If you run a program in a \gls{terminal}, then the text that a process writes to its \gls{stdout} appears in the console.%
